# Typing Expert

# Concepts used

-   Menu UI
    - Text Mesh Pro
    - Buttons
    - Panels
    - Render Modes

- Scene Change
- Game Pausing (Time Stoping)


# Game Play

-   Press on start to play the game.
-   Input the words through keyboard to earn score.
-   Don't do mistake, and your score multipier will increase. 
-   Press escape to pause the game.
